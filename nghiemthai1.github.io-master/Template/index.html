Object Follower

This project implemented a PID controller for a sumo robot object-follower. The sumorobot was designed to meet the Rowan Prof Bots sumo robot competition, and was implemented to have minimal overshoot, minimal steady-state error and a low settling-time. Ultimately, the sumo bot was able to follow a moving object at a fixed distance and account for turning, always stopping directly facing the object.

C++, System Control
